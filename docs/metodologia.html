METODOLOGÍA

La página web que aquí se expone ha sido creada mediante un duro procedimiento: Tuve que transformar los archivos de las prácticas realizadas durante el curso a lenguaje html, para lo que utilicé la herramienta pandoc.

También tuve que crear una carpeta docs visible en el repositorio. Para ello, creé archivos nuevos en formato HTML empleando el editor texto nano. En la mencionada carpeta docs, aparecen todos los archivos html que se deben visualizar:

  • Cabecera.html
  • Las cuatro prácticas correspondientes
  • Pie.html
  • Metodología.html Además del css, el js y el Index.html.

He modificado el estilo de la pagina web a través de bootstrapp También modifiqué la cabecera cambiando el título, el nombre del autor, y la aplicación usada Por último, conseguí enlazar todas las prácticas cambiando el index y la cabecera Además, arreglé la ruta de las imágenes y añadí el enlace de datawrapper a los gráficos a través de frame

Para q esto apareciese en formato web, accedí desde el panel de Settings de Github a Github Pages, donde seleccioné docs. En lo relativo a la página web, con el propósito de que se viera una barra superior tipo menú, edité la plantilla de Bootstrap.

Como conclusión extraigo lo difícil que ha sido para mí poder entregar este trabajo y el esfuerzo que me ha supuesto para llegar a este nivel, aunque soy consciente de que no es muy elevado.
